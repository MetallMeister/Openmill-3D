# Openmill-3D

Openmill-3D は、3Dプリンタで製作可能なパーツを使用した、初心者向けオープンソースCNCマシンプロジェクトです。  
現在は試作・開発段階であり、仕様や構造は今後変更される可能性があります。

## 特徴

- 3Dプリンタで出力できる部品を多用
- 卓上サイズのコンパクト設計
- オープンソース（※商用利用不可）
